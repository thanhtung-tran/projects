Data cleaning in SQL

The objective of this project was to prepare a Nashville housing dataset for accurate market analysis. The dataset had two main issues: inconsistent date formats and missing property addresses. I addressed these by converting all dates to a consistent 'YYYY-MM-DD' format using SQL date functions and filling in missing addresses through complex SQL joins with auxiliary data sources. The result was a cleaned dataset that allowed for precise analysis of housing market trends, demonstrating my skills in data manipulation and attention to detail.
